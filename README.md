# excel_text_finder

# Example
## Example directory
- root
  - data
    - ex1.xlsx
    - ex2.xlsx
  - src
    - main.py
    - text_finder.py

## Arguments
- arg1 : path (file path or directory path)
- arg2 : name of sheet
- arg3 : target text to find

## Example
`python src/main.py data/ex1.xlsx Sheet1 MyText`

`python src/main.py data Sheet1 MyText`
